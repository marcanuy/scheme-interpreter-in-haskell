# scheme-interpreter-in-haskell

Playing around with [Write Yourself a Scheme in 48 Hours Haskell Tutorial](https://en.wikibooks.org/wiki/Write_Yourself_a_Scheme_in_48_HoursURL ).

Writing a fully-functional Scheme interpreter that implements a subset of [R5RS Scheme](http://www.schemers.org/Documents/Standards/R5RS/HTML/).

## Uses ##

+ Haskell's I/O
+ mutable state
+ dynamic typing
+ error handling
+ parsing features
